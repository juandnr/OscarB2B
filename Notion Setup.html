01

Crear el Worker en Cloudflare

Gratis · ~5 minutos · No necesitas tarjeta de crédito

1. Ve a dash.cloudflare.com y crea una cuenta gratuita.

2. En el menú lateral, ve a Workers & Pages → Create.

3. Selecciona "Create Worker", dale cualquier nombre (ej: oscar-notion-proxy).

4. Borra todo el código que aparece y pega el código de abajo:

// Oscar B2B — Notion CORS Proxy Worker
// Cloudflare Workers (gratis, dash.cloudflare.com)

export default {
  async fetch(request, env) {
    const cors = {
      'Access-Control-Allow-Origin': '*',
      'Access-Control-Allow-Methods': 'GET, POST, PATCH, DELETE, OPTIONS',
      'Access-Control-Allow-Headers': 'Content-Type, Authorization, Notion-Version',
    };

    if (request.method === 'OPTIONS') {
      return new Response(null, { headers: cors });
    }

    const url = new URL(request.url);
    const target = 'https://api.notion.com' + url.pathname + url.search;
    const hasBody = ['POST', 'PATCH', 'PUT'].includes(request.method);

    const response = await fetch(target, {
      method: request.method,
      headers: {
        'Authorization': request.headers.get('Authorization') || '',
        'Notion-Version': '2022-06-28',
        'Content-Type': 'application/json',
      },
      body: hasBody ? request.body : null,
    });

    const data = await response.text();

    return new Response(data, {
      status: response.status,
      headers: { ...cors, 'Content-Type': 'application/json' },
    });
  }
};

5. Haz clic en "Deploy". Copia la URL del Worker que aparece (ej: https://oscar-notion-proxy.tusubdominio.workers.dev). La necesitarás después.

02

Crear la Integración en Notion

Genera el token que conecta las herramientas con tu workspace

1. Ve a notion.so/my-integrations.

2. Haz clic en "+ New integration".

3. Nómbrala "Oscar B2B Tools", selecciona tu workspace y haz clic en Submit.

4. Copia el Internal Integration Token (comienza con secret_). Guárdalo en un lugar seguro.

Importante: Este token es como una contraseña. No lo compartas públicamente.

Conectar las databases a la integración: abre cada database, ve a los 3 puntos (···) → Add connections → Oscar B2B Tools.

Obtener el ID de cada database: el ID está en la URL de la database. Ejemplo: notion.so/workspace/abc123def456...?v=... — los primeros 32 caracteres después del último / y antes del ?.

04

Configurar en el CRM y Dashboard

Último paso — menos de 1 minuto

1. Abre el CRM o el Dashboard.

2. Haz clic en el ícono ⚙ (arriba a la derecha).

3. Ingresa:

  • Worker URL — la URL del Worker que copiaste en el paso 1
  • Token de Integración — el secret_... del paso 2
  • ID Base Datos CRM — el ID de la database CRM del paso 3
  • ID Base Datos KPIs — el ID de la database KPIs del paso 3

4. Haz clic en "Probar conexión" — deberías ver "✓ Conexión exitosa".

5. Guarda y ya está. Desde ahora, el botón "Notion" en la barra superior sincronizará tus datos.

Nota: La configuración se guarda en tu navegador. Si accedes desde otro computador, repite el paso 4 allí también.
